# TrabalhoJogadores
Contexto de futebol
Trabalho de Introdução a Programação/Autoria Web.
Contexto: Futebol. Elemento: Jogadores.
Futebol é um esporte coletivo mundialmente popular, jogado por duas equipes de 11 jogadores (incluindo um goleiro) em um campo retangular, cujo objetivo é chutar ou cabecear uma bola para dentro da meta adversária, usando principalmente os pés e o corpo, mas não as mãos (exceto o goleiro em sua área). Existe vários campeonatos com muita relevância em todo mundo. Jogadores sao aqueles responsáveis em jogar a bola dentro da meta adversária.
Esse site tem o objetivo de criar um perfil de um jogador, e vai esta pedindo 5 dados sobre esse jogador sendo esses dados:
Time; Gols; Assistências; Idade; Nacionalidade.
